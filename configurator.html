Setup guide
  1. In OBS, click + in the Sources panel and choose Browser
  2. Check Local file, then browse to your downloaded .html file
  3. Set width and height to 512 × 512
  4. Check Shutdown source when not visible (optional but recommended)
  5. Click OK — your overlay is live
Make sure your pad is connected and recognised by your OS before launching the scene.
  1. In Streamlabs, click + in the Sources panel and choose Browser Source
  2. Toggle Local file on, then browse to your downloaded .html file
  3. Set width and height to 512 × 512
  4. Click Done
Streamlabs uses the same Chromium browser source as OBS — the overlay behaves identically.
  1. In vMix, click Add Input → Web Browser
  2. Set the URL to the full local path of your file, prefixed with file:///
    e.g. file:///C:/overlays/pioc-overlay.html
  3. Set width and height to 512 × 512
  4. Click OK and add the input to your mix
vMix's browser input supports the Gamepad API — no extra configuration needed.
  1. In XSplit Broadcaster, click Add Source → Webpage
  2. Enter the full local path prefixed with file:///
    e.g. file:///C:/overlays/pioc-overlay.html
  3. In the source properties, set resolution to 512 × 512
  4. Click OK
XSplit's Chromium-based webpage source supports the Gamepad API.
