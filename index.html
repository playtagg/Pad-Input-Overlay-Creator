Before you start

Requirements

Your pad

  • Must be recognized as a gamepad/HID joystick by your OS
  • FSR pads running ADP firmware work out of the box
  • Any DIY pad using HID-compatible firmware should work

Your software

  • Works in any Chromium-based browser locally — just open the HTML file
  • OBS Studio, Streamlabs, XSplit, or any software that supports browser sources for streaming
  • Browser source must support the Gamepad API (all modern Chromium-based browser sources do)
How it works This tool uses the browser's built-in Gamepad API as the bridge between your physical pad and what you see in the overlay. When you press a panel, your pad sends a button signal through HID, the browser picks it up via the Gamepad API, and the overlay reacts. No extra software or drivers needed beyond what makes your pad show up as a gamepad in the first place.

Getting started

Setting up your overlay

01

Create your overlay

Use the configurator to pick a pad style, assign your inputs, and customize your colors. Hit Export to download the HTML file.

02

Connect your pad

Make sure your pad is connected and recognized as a gamepad by your OS before opening OBS.

03

Add a browser source

In OBS, click + in the Sources panel and choose Browser. Check Local File and select your downloaded HTML file.

04

Set the dimensions

Set width and height to 512 × 512. Position it wherever you want in your scene.
