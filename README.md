# PIOC — Pad Input Overlay Creator

Create custom dance pad input overlays for use in OBS, Streamlabs, vMix, XSplit, or any Chromium-based browser source.

**[→ playtagg.github.io/pad-input-overlay-creator](https://playtagg.github.io/pad-input-overlay-creator/)**

## What it does

Configure a pad overlay in a few steps — pick a style, map your inputs by stepping on your pad, set your colors, and export a self-contained HTML file. Drop it in as a browser source and you're done.

## Styles

- **Simple** — dark grid layout with arrow icons and FSR bar indicators
- **Modern** — ultra-minimal panels, no decoration
- **FSR Mini Pad** — detailed octagonal pad styled after the Rhythm Technologies FSR Mini Pad

## Requirements

- Pad must be recognized as a gamepad/HID joystick by your OS
- FSR pads running ADP firmware work out of the box
- Any modern browser or Chromium-based browser source
